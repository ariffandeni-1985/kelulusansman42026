📌 Cara Hosting Situs Ini Agar Bisa Diakses Siswa

Situs ini adalah file HTML tunggal (index.html) yang bisa dihosting gratis menggunakan layanan berikut:
🌐 Opsi 1 — Netlify (Rekomendasi, Gratis)
  1. Buka netlify.com dan daftar akun gratis
  2. Di dashboard, klik "Add new site" → "Deploy manually"
  3. Drag & drop file index.html ke area upload
  4. Situs langsung aktif dengan URL seperti random-name.netlify.app
  5. Bisa ubah nama domain menjadi nama sekolah (misalnya: kelulusan-sman4tjb.netlify.app)
✅ Gratis selamanya untuk situs statis. Tidak perlu bayar hosting.
🌐 Opsi 2 — GitHub Pages (Gratis)
  1. Buat akun di github.com
  2. Buat repository baru, upload file index.html
  3. Masuk ke Settings → Pages → Source: main / root
  4. URL: username.github.io/nama-repo
⚠ Catatan Penting Tentang Data

Situs ini menggunakan storage lokal Claude Artifact untuk menyimpan data. Jika dihosting ke Netlify/GitHub Pages, data siswa yang diunggah melalui panel admin tidak akan tersimpan secara permanen karena tidak ada database backend.

Solusi: Untuk hosting mandiri, tambahkan backend sederhana (misalnya Firebase Firestore atau Google Sheets API) atau ganti mekanisme penyimpanan dengan localStorage di browser.

⚡ Hubungi pengembang untuk membantu integrasi database agar data siswa bisa disimpan secara permanen di server.
